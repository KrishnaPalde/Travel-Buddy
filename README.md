 INTRODUCTION
---------------

Travel Buddy, a travel companion, but not in this case. Travel Buddy referes to an application which helps you plan your itinerary. 


 Technology Stack
--------------------
1. Visual Basic (.Net)
2. Bunifu UI Development Framework for VB.Net
3. MySQL Database
